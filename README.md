# RightKnight

### Описание
Утилита предназначена для подсчёта шахматного перформанса по сыгранным играм на платформе lichess.org

### Локальная установка

    git clone git@github.com:ppeter777/RightKnight.git
    cd RightNight
    make install

### Быстрый запуск с помощью Make:

    make build — собрать проект.
    make run — запустить сервер (остановка через Ctrl + C).
    make clean — удалить скомпилированные файлы и сбросить кэш сборки.
    make stop — завершить фоновые процессы Gradle (освободить RAM)..

в браузере перейти на http://localhost:8080/performance
